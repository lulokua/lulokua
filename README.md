## 📚 Publications / 论文发表

### Watermelon Maturity Assessment Technology Based on Machine Vision  
### 基于机器视觉的西瓜成熟度评估技术

**Arxiv · 2024**

### 中文摘要

本文使用计算机视觉与深度学习方法判断西瓜是否成熟。  
核心思路是使用预训练的 **VGG16** 模型提取西瓜图像特征，并结合 **数据增强**、**Dropout** 和 **L2 正则化** 等方法，缓解小样本训练带来的过拟合问题。

### English Abstract

This paper uses computer vision and deep learning methods to determine whether a watermelon is ripe.

The core idea is to use a pre-trained **VGG16** model to extract features from watermelon images, and then apply techniques such as **Data Augmentation**, **Dropout**, and **L2 Regularization** to reduce overfitting caused by small-sample training.
